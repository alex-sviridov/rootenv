# Contmgr Service ŌĆö Implementation Plan

## Context

SSH private keys are currently mocked in `servers.connection` by a PocketBase hook. This plan introduces a real `contmgr` Go service that spins up Docker containers, generates SSH keypairs, and stores private keys encrypted in the `keys` table. The relay decrypts them at connection time using a per-attempt `secret` fetched by the user.

**Rename:** `servers` collection ŌåÆ `assets` throughout (PocketBase, hooks, relay, frontend). Each asset has a `platform` field (`container` now; `vm` planned).

---

## Key Design Decisions

| Concern | Decision |
|---|---|
| Secret field name | `keys.secret` (32-char alphanumeric, auto-generated by PocketBase) |
| Encryption | AES-256-GCM; key = SHA-256(`keys.secret`); stored as `base64(12-byte-nonce \|\| ciphertext)` |
| SSH keypair | Ed25519, generated in-memory |
| Docker image | Per-asset `image` field in YAML; initial value `hermsi/alpine-sshd` |
| SSH user | Per-asset `ssh_user` field in YAML |
| Public key injection | Docker exec into running container after start |
| Port assignment | `net.Listen("tcp", ":0")` ŌåÆ capture port ŌåÆ close ŌåÆ publish to container |
| Contmgr trigger | Poll PocketBase every 5s |
| Relay auth message | `<pb_token>\n<keys.secret>` (newline-separated, single WS frame) |
| Service account | `svc_contmgr@contmgr.local`, `svc_role="contmgr"` |
| Asset configuration | `assets.configuration` JSON `{"platform":"container","id":"<containerID>"}` ŌĆö extensible for future platforms |

## Data Structures

**`attempts.environment` (YAML ŌåÆ JSON, contmgr reads):**
```json
{
  "duration": 90,
  "servers": [
    {"name": "server-0", "image": "hermsi/alpine-sshd", "ssh_user": "lab", "cpu": "1", "memory": "512MB", "disk": "5GB"}
  ]
}
```
(`servers` key stays in YAML env spec ŌĆö `assets` rename is for the PocketBase collection only.)

**`assets.connection` after contmgr writes:**
```json
{"host": "172.17.0.1", "port": 32768, "user": "lab"}
```

**`assets.configuration` after contmgr writes:**
```json
{"platform": "container", "id": "<containerID>"}
```

**`keys` record after contmgr writes:**
```
secret:        "Xk9mP2..."  (32-char, PocketBase auto-generates)
key_encrypted: "<base64(12-byte-nonce || AES-GCM ciphertext of Ed25519 PEM)>"
```

**Relay WS auth message (first frame):**
```
<pocketbase_jwt_token>\n<keys.secret>
```

### `keygen.go`
```go
// GenerateKeypair returns (privateKeyPEM, authorizedKeysLine, error).
func GenerateKeypair() ([]byte, []byte, error)
// Ed25519: crypto/ed25519 + golang.org/x/crypto/ssh
// privateKeyPEM: ssh.MarshalPrivateKey
// authorizedKeysLine: ssh.MarshalAuthorizedKey

// EncryptPrivateKey: AES-256-GCM; key=SHA-256(secret); returns base64(nonce||ciphertext).
func EncryptPrivateKey(privateKeyPEM []byte, secret string) (string, error)
```

### `docker.go`
```go
type ContainerParams struct {
    Image     string
    SSHUser   string
    PublicKey []byte   // authorized_keys content
    CPU       string   // "1"
    Memory    string   // "512MB" ŌĆö parsed to bytes
}

// CreateAndStart returns (containerID, hostPort, error).
// Steps:
//   1. net.Listen(":0") ŌåÆ grab port number ŌåÆ close listener
//   2. ContainerCreate with SSH_USERS env, port binding :22ŌåÆhostPort
//   3. ContainerStart
//   4. ExecCreate + ExecStart: write public key file inside container:
//      sh -c "mkdir -p /conf.d/authorized_keys && printf '%s' '<pubkey>' > /conf.d/authorized_keys/<ssh_user>"
func (d *DockerClient) CreateAndStart(ctx context.Context, p ContainerParams) (string, int, error)

// Remove stops and removes a container by ID.
func (d *DockerClient) Remove(ctx context.Context, containerID string) error
```

Wait for SSH readiness: `net.DialTimeout("tcp", host:port, 1s)` ├Ś 30 attempts.

### `contmgr.go`
```go
func (p *Contmgr) ProvisionAsset(ctx context.Context, asset Asset) error
// 1. GetAttempt(asset.Attempt) ŌåÆ parse environment.servers ŌåÆ find def where name==asset.Name
// 2. GenerateKeypair() ŌåÆ privKeyPEM, pubKeyLine
// 3. docker.CreateAndStart(ContainerParams{def.Image, def.SSHUser, pubKeyLine, def.CPU, def.Memory})
//    ŌåÆ containerID, hostPort
// 4. Wait for SSH ready (poll TCP)
// 5. GetKeysByAttempt(asset.Attempt) ŌåÆ keysRecord
// 6. EncryptPrivateKey(privKeyPEM, keysRecord.Secret) ŌåÆ ciphertext
// 7. PatchKeys(keysRecord.ID, {"key_encrypted": ciphertext})
// 8. PatchAsset(asset.ID, {
//      "connection":    {"host": cfg.hostIP, "port": hostPort, "user": def.SSHUser},
//      "configuration": {"platform": "container", "id": containerID}
//    })
// 9. PatchAsset(asset.ID, {"state": "provisioned"})

func (p *Contmgr) DecommissionAsset(ctx context.Context, asset Asset) error
// 1. Parse asset.Configuration ŌåÆ extract containerID (field "id")
// 2. docker.Remove(containerID)
// 3. PatchAsset(asset.ID, {"state": "decommissioned"})

func (p *Contmgr) RunOnce(ctx context.Context) error
// a. ListProvisioningAssets() ŌåÆ for each, spawn goroutine ŌåÆ ProvisionAsset
// b. ListPendingDecommissionCommands() ŌåÆ for each:
//      PatchCommand(cmd.ID, {"status":"running"})
//      GetAsset(cmd.Asset) ŌåÆ DecommissionAsset
//      PatchCommand(cmd.ID, {"status":"done"})
// Use errgroup for concurrency; log errors but don't abort entire cycle on one failure
```

### `main.go`
```go
// load config ŌåÆ auth with PocketBase (retry 5├Ś with backoff) ŌåÆ init Docker client
// ticker loop: RunOnce every cfg.pollInterval
// graceful shutdown: signal.NotifyContext(SIGINT, SIGTERM) ŌåÆ drain in-flight work
```

## Phase 7 ŌĆö Relay: decrypt key at connection time

**Goal:** Relay stops using a static private key file; instead decrypts per-connection from `keys` table.

```go

// base64-decode ŌåÆ split [0:12]=nonce, [12:]=ciphertext ŌåÆ AES-256-GCM decrypt
// ŌåÆ ssh.ParsePrivateKey ŌåÆ ssh.Signer
func decryptPrivateKey(keyEncrypted, secret string) (ssh.Signer, error)
// Key derivation: sha256.Sum256([]byte(secret)) ŌĆö must match contmgr exactly
```

### `relay/handler.go`


## Phase 8 ŌĆö Frontend: fetch secret and update WS auth

**Goal:** Frontend sends `token\nsecret` as the first WS frame.

   `GET /api/collections/keys_userview/records?filter=(attempt='{attemptID}')` with user auth token ŌåÆ extract `record.secret`

3. On WebSocket open, send `pb.authStore.token + "\n" + secret` instead of just the token

**Note:** `keys_userview` structure:

## Scope Boundaries

- No TLS or SSH host key pinning ŌĆö `InsecureIgnoreHostKey` is correct for ephemeral internal containers
- No image pre-pull ŌĆö assume `hermsi/alpine-sshd` is available on host
- No multi-host Docker ŌĆö single host via docker.sock
- No start/stop/restart commands in contmgr ŌĆö only provision and decommission
- Contmgr is stateless ŌĆö all state lives in PocketBase
